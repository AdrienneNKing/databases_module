6)
 >
Greater than, used when comparing 2 values ex. 8 > 5

<
Less than, used when comparing 2 values ex. 2 < 10

=
Equal to, used when comparing 2 values ex. name = 'Adrienne'

<=
Less than or equal to, used when comparing 2 values ex. 9 <= 10

>=
Greater than or equal to, used when comparing 2 values ex. 10>=10

!=
Not equal to, used when comparing 2 values 5 != 6


<>
Greater than or less than, used when comparing 2 values ex. price <> 10.00

LIKE
Used to pattern match a string ex. category LIKE '&travel';

BETWEEN
Implemented like "a BETWEEN x AND y", this is equivalent to "a >= x AND a <= y" ex. SELECT age from people WHERE age BETWEEN 5 AND 6;

IS DISTINCT FROM
Used as "expression IS DISTINCT FROM expression", this should be used in place of <> when one of the inputs could be null. ex. Cat IS DISTINCT FROM null

AND
Requires both expressions to be true for the row to be returned. ex. SELECT breed from dogs WHERE age = 5 AND name = 'Sadie';

OR
Requires one of the expressions to be true for the row to be returned ex. SELECT breed from dogs WHERE age = 5 OR age = 7;

NOT
Requires the expression to be false for the row to be returned SELECT breed from dogs WHERE NOT age = 7;
